# Strava my stats

## Prerequisite installation

```shell
$ sudo pip3 install pystrava
```

## Configuration

- Maintain the Strava credentials in `.credentials.properties`
- Maintain the parameters in the `config.properties` file

## Execution

```shell
$ python3 my_strava_stats.py
```
